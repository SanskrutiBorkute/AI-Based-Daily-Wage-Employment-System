# 📸 AI-Based Daily Wage Employment System – Project Gallery

This gallery showcases the major screens of the **AI-Based Daily Wage Employment System (KaamSetu)**. The application connects daily wage workers with employers using AI-powered job matching, wage recommendations, multilingual support, and smart assistance.

## Gallery Summary

The AI-Based Daily Wage Employment System (KaamSetu) offers a complete digital platform for connecting employers with daily wage workers. The project combines Artificial Intelligence, multilingual support, AI wage prediction, smart job matching, notifications, and an easy-to-use mobile-friendly interface to simplify the hiring process and improve employment opportunities.
